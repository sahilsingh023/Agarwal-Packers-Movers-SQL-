# Agarwal-Packers-Movers-SQL-
This project focuses on analyzing the day-to-day operations of Agarwal Packers & Movers using MySQL.
It solves real business problems like delayed deliveries, vehicle utilization, revenue tracking, customer retention, and branch performance through SQL queries.

The goal of this project is to demonstrate how SQL can be used for logistics analytics, operational optimization, and decision-making.
# Schema Structure-
The project contains the following tables:

customers → customer details

bookings → shipment booking information

shipments → delivery tracking

payments → revenue & payment status

vehicles → fleet management

drivers → driver details

driver_attendance → driver reporting time

inventory → packaging material stock
# Business Problems Solved-
This project answers key operational questions:

Identify unassigned bookings

Detect delayed deliveries

Find duplicate customers

Track branch-wise performance

Analyze revenue by city & month

Monitor vehicle utilization & idle vehicles

Identify low inventory alerts

Find top customers by revenue

Measure on-time delivery %

Detect overloaded vehicles
